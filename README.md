## 📖 About

This repository documents my hands-on learning in **Web Application Security**, including vulnerability analysis, attack methodologies, payload collections, lab notes, and secure coding practices.

The objective is to build a structured cybersecurity knowledge base while developing practical skills through platforms such as PortSwigger Web Security Academy, OWASP, and TryHackMe.
